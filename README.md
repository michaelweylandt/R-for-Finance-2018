# R-for-Finance-2018

This repository contains materials for my 2018 "R for Finance" short course. 

The materials are for download as Jupyter notebooks (to be run with the "IR"
kernel -- https://irkernel.github.io/) or experimentally as Binder notebooks
(http://mybinder.org). 

To launch Binder, click the badge below: 

[![Binder](https://mybinder.org/badge.svg)](https://mybinder.org/v2/gh/michaelweylandt/R-for-Finance-2018/master)
